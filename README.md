# Study Cognito

Study Cognito is a study planner built for hackers and developers who want structure while learning technical skills.

Users enter their learning goal, available study hours, and duration. The app generates a clear, day-wise study plan focused on practical learning instead of random resources.

## Why Study Cognito?
Many beginners feel lost about *what to study next*. Study Cognito helps reduce confusion by breaking learning goals into manageable daily tasks.

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Open `index.html` in a browser and enter your study details.
